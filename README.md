# UML
A python script, that creates simple Uml diagrams in  ASCII art from user inputs.

# usage
Clone the repository:

```bash
git clone https://github.com/Karturbis/UML.git
```

Go into the directory of the program:

```bash
cd UML
```

### Run the script:

#### Unix:
```bash
python3 uml-diagrams.py
```

#### Windows:
```bash
python uml-diagrams.py
```
